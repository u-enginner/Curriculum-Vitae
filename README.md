## 強み
　私の長所はスピード力です。今まで参画してきた現場でもプロパー、他の協力会社様を問わずにコミュニケーションや手空き中に業務効率を上げられる手順書やマクロ作りなど率先した行動を心掛け、現場の雰囲気作りに貢献できるよう意識しておりました。  
開発業務では計画より前倒しで作業を完了させることを心掛けておりました。その際、ミスや漏れがない様に確認フローを自身で定義してPDCAを素早く回すことを意識しました。クライアントからの要望や質問にも素早く対応し、信頼関係の向上にも役立ったと感じております。  
　社内で実施される任意参加の技術向上目的の開発研修に積極的に参加し、普段の業務で触れられない技術や言語で開発することで日々新しい知識について勉強しています。  
　まずは技術をしっかりと磨き即戦力として貴社へ貢献できる事を第一に、将来的にはプロジェクトを技術面も含めて総合的に把握、検討できる役割を担当できることを目指しております。  
　今後も一生を通じて様々な事を学び続け、チャレンジして参ります。よろしくお願いいたします。
